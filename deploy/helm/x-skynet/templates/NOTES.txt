Then open: http://localhost:3000

==========================================================
 CONFIGURATION
==========================================================

To update your deployment:

  helm upgrade {{ .Release.Name }} x-skynet/x-skynet \
    --set secrets.ANTHROPIC_API_KEY=<your-key> \
    --set replicaCount=2 \
    -n {{ .Release.Namespace }}

To uninstall:

  helm uninstall {{ .Release.Name }} -n {{ .Release.Namespace }}

==========================================================
 IMPORTANT: Set your API keys before using X-Skynet!
==========================================================

  kubectl create secret generic {{ include "x-skynet.fullname" . }}-secrets \
    --from-literal=ANTHROPIC_API_KEY=<your-anthropic-key> \
    --from-literal=OPENAI_API_KEY=<your-openai-key> \
    --from-literal=REDIS_URL=redis://redis:6379 \
    -n {{ .Release.Namespace }} --dry-run=client -o yaml | kubectl apply -f -
