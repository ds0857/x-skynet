# Hello World Example

This example demonstrates:
- Creating an agent
- Running a simple task
- Printing the result

## Run

1) Install deps at repo root

   pnpm install

2) Build

   pnpm --filter @xskynet/example-hello-world build

3) Run

   pnpm --filter @xskynet/example-hello-world start

## Expected Output

Result: { text: 'Hello, world!' }
