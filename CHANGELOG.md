# Changelog

## [0.1.0] - 2026-02-23

### Added
- Core runtime engine with event-driven architecture
- Plugin system: plugin-claude, plugin-http, plugin-shell, plugin-telegram, plugin-memory
- TypeScript SDK (sdk-js) and Python SDK (sdk-py)
- CLI with `xskynet run`, `xskynet dev`, `xskynet logs`
- DAG Run Viewer (React + Mermaid.js)
- VitePress documentation site
- GitHub Actions CI/CD pipeline
- End-to-end test suite (44 tests)
- Example workflows: hello-world, research-agent
